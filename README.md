# WCAG Checker

> Fast, developer-first accessibility testing for WCAG 2.2

WCAG Checker is an open-source CLI tool that helps developers detect and fix accessibility issues in their websites. It scans pages using real browsers and evaluates them against WCAG standards, generating clear and actionable reports.

---

# WCAG Checker – First Release Scope

List of rules and guidelines intended to be covered in the first release:

### 1.1 Text Alternatives
